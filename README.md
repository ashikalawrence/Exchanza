# Exchanza 📚

Exchanza is an AI-powered smart book exchange platform built using React + Firebase.

## Features
- Google Authentication
- Smart Book Matching
- Wishlist System
- Real-time Exchange Requests
- Book Circles Community
- AI Recommendations
- Responsive UI
- Firebase Firestore Database
- Real-time Notifications

## Tech Stack
- React.js
- Firebase
- Firestore
- Firebase Authentication
- Vite
- CSS
